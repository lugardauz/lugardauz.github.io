Início Testemunhos Festas de Calendário Sugestões Típicas Mezinhas Orações


- Chá de marcela -> para a febre

- Chá de urtigas -> para a circulação

- Colar de alhos -> para as lombrigas

- Aguardente queimada -> constipações

- Hortelã debaixo do travesseiro -> para matar as "bichas"(lombrigas)

- Chá de pelicão -> para infecções da bexiga

- Chá de nêveda -> para as "bichas"(lombrigas)

- Chá de hortelã e salvia -> para o fígado

- Chá de cidreira -> para o estômago, má disposição

- Chá de tília -> para os nervos

- Chá de pau de canela -> para as dores menstruais

- Chá de carqueja -> para a tosse

- Chá de pés de cereja e barbas de milho -> para os rins

- Xarope de cenoura -> para a tosse
(Deixam-se a marinar de um dia para o outro, cenouras cortadas às rodelas cobertas de açucar)

- Chá de folha de laranjeira -> para a tosse

- Urtigar as pernas e o pescoço -> para as dores de dentes

- "Bosta"(excrementos) de vaca -> para atenuar as dores de queimaduras

- Teias de aranha -> para estancar o sangue

- Pó do caruncho da madeira -> para as infecções do "xixi" do bebés

- Aguardente, água, limão e açucar -> para a "diarreia"

- Água de "clipe"(eucalipto) -> para as constipações e para lavar as "fridas" infeccionadas

- "Chuchinhos" de açucar -> para os bebés não chorarem

- Refresco de aguardente, água e açucar -> para matar a sede

- Quando um miúdo estava gago ou não falava (entaramelava a língua), arranjava-se um fole (pele de cabra seca), metia-se o miúdo lá dentro e os pais iam com ele de casa em casa e faziam esta reza:
"Quem dá a esmola
Ao menino do fole
Que quer falar
E não pode"
Entravam por uma porta das casas e tinham de sair por outra

- "Talhar as Dadas"
Para aliviar as dores de cabeça, causadas pelo «mal de inveja», talhavam-se as dadas com a seguinte reza:
"(O nome da pessoa a quem se ia talhar as dadas) de Jesus
Se tens dadas
Deus as bote
A quem as botou
Com o poder de Deus
E da Virgem Maria
Um Pai Nosso
Com uma Avé Maria"
À medida que se rezava iam-se deitando brasas numa tigela com água (dez brasas), por cada Pai Nosso uma brasa. No fim, a última brasa deitava-se para trás das costas.
Se as brasas fossem ao fundo da água a pessoa tinha dadas(mau olhado).

- "Talhar o coxo"
Quando apareciam muitas borbulhas no pescoço(cocho) fazia-se a seguinte reza:
"Faca que cortas o pão
Corta este Tubarão
Corto-te a cabeça
Corto-te o rabo
Com este navalhão
Sapo, sapão
Cabra, cabrão
Toupa, toupeirão
Corto-te a cabeça
Corto-te o rabo
Com este navalhão
Com o poder de Deus
E da Virgem Maria
Um Pai Nosso e uma Avé Maria"
À medida que se rezava iam-se fazendo cruzes no pescoço com uma faca.

- Quando Se abria um pulso ou um pé, fervia-se uma cafeteira de barro de água. Deitava-se a água numa bacia e punha-se o pé ou a mão magoada a apanhar o vapor da água na bacia("pôr o pé a cozer").
Ao mesmo tempo ia-se rezando:
"1ªpessoa - Eu te cozo,
2ªpessoa(magoada) - carne aberta e fio torto
1ªpessoa - É mesmo isso que eu cozo."

- Quando se via uma estrela cadente costumava-se dizer:
"Assim corra a minha alminha p'ro Céu!"
